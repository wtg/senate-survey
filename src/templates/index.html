Senate Surveys

The Rensselaer Union Student Senate's Web Technologies Group has created a surveying platform that allows the student body at RPI to create their own custom-made survey (in development).

Start Survey Create your own survey

Completely anonymous

While this survey requires that students log in using a valid RCS user ID, the survey itself is completely anonymous. The login is merely used to authenticate that survey participants are affiliated with RPI, and all personally-identifiable user information is deleted after the session is complete. An irreversible hash of your RCS ID is recorded to ensure that the survey is taken only once, and this hash is not linked to your survey response. Furthermore, the open-source code for the survey platform can be found on GitHub.
